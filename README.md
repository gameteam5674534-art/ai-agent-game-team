# ai-agent-game-team
Multi-agent AI team that collaboratively guides and assists users in designing and developing games.
## 🛠️ 3步極速上手 (Quick Start)

我們將繁瑣的 AI 設定簡化到了極致！只需跟著以下三個步驟，就能在你的終端機 (CLI) 中喚醒專屬的 AI Agent：

### 📋 事前準備
在開始之前，請確保你的電腦已安裝：
* [Git](https://git-scm.com/)
* [Node.js](https://nodejs.org/) (建議 v18 以上) 或 [Python 3.9+](https://www.python.org/) *(註：請根據你們專案的語言二選一，把沒用的刪掉)*

---

### 1️⃣ 複製專案與安裝依賴 (Clone & Install)
打開你的終端機（Terminal 或命令提示字元），依序輸入以下指令：

```bash
# 1. 將專案複製到你的電腦本地端
git clone [https://github.com/](https://github.com/)[你的帳號]/[專案名稱].git

# 2. 移動到專案資料夾
cd [專案名稱]

# 3. 安裝專案所需的套件
npm install  # 如果你們是 Python 專案，請改成：pip install -r requirements.txt
